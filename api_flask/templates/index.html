SentimentStream

Pipeline de análisis de sentimientos de extremo a extremo. Procesa comentarios en tiempo real mediante Spark Structured Streaming, clasifica su sentimiento con Naive Bayes y expone los resultados a través de una API REST conectada a MongoDB Atlas.
